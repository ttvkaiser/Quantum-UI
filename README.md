## Usage

### Loadstring:
```lua
local quantumui = loadstring(game:HttpGet("https://example-domain.com/quantum-ui.lua"))()
```

### Add Main window:
```lua
local window = quantumui:CreateWindow{
    Title = "Quantum UI Example",
    SubTitle = "by ttvkaiser",
    Size = UDim2.fromOffset(600, 600),
    Theme = "QuantumMain"
}
```
